Gravatar Integration README

Revision 2008-03-11

* Installation * 

Install the module at /admin/build/modules
Make sure user picture support is enabled at /admin/user/settings#edit-user-pictures-0-wrapper 
Enable gravatar suport at /admin/user/gravatar

Enjoy

* User settings *

Each user may choose to use the gravatar.com service, default is off

If the email field is left blank, the account email address is used
Alternatively, the authenticated user may enter a different 
address to use with the gravatar.com service
